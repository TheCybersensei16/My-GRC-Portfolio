## Key Findings & Recommendations
1. **Critical** – Privileged access rights not fully enforced (A.8.2)  
   Finding: Admin accounts shared without MFA.  
   Recommendation: Implement PAM tool + quarterly access review.

2. **Critical** – Data Subject Access Request (DSAR) process missing (DPDPA Sec 6)  
   Finding: No documented process for data principal requests.  
   Recommendation: Create DSAR playbook + 72-hour response SLA.

3. **High** – Third-party vendor risk assessment incomplete (A.15.1)  
4. **High** – Logging & monitoring gaps for production systems (A.12.4)  
5. **High** – Incident response plan not tested (A.16.1)  
6. **High** – Cryptographic controls not uniformly applied (A.10.1)  
7. **High** – Employee security awareness training not tracked (A.8.3)
