# DPDPA DSAR Playbook – PaySecure India Pvt Ltd
**Version:** 1.0 | **Date:** 04 April 2026

### What is a DSAR?
Any request from a Data Principal for Access, Correction, Erasure, or Grievance.

### Step-by-Step Process (72-hour SLA)
1. **Receive Request** (email / form / phone) → Log in DSAR Register (Excel)
2. **Verify Identity** (within 24 hrs) – Ask for email + any one more identifier
3. **Acknowledge** – Send auto-reply with ticket number and expected timeline
4. **Process Request** (within 48 hrs)
   - Access → Export data in readable format
   - Correction → Update record
   - Erasure → Delete + confirm
5. **Respond** – Email final response + evidence
6. **Close ticket** & update DSAR Register
